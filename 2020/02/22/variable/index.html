JavaScript語法

這次要介紹的語法主要是宣告變數:

  1. let
  2. var
  3. const

在介紹上述3個宣告之前我們要先介紹何謂作用域（Scope）

我先慣把作用域分成全域變數(Global Variable)和區域變數(Local Variable)兩種

全域變數(Global Variable)

假設我們的程式主要有3個功能

A-function(求圓形面積)

B-function(求球形面積)

C-function(求圓錐面積)

3個功能互相獨立而且不相依

3個方程式計算時必須共同要用到Pi=3.14159這個變數，而Pi就是所謂全域變數(Global Variable)

全域變數的特性用一句話來表示就是：牽一髮而動全身

以下圖為例：

Pi=3.14159 變成 Pi=3時，下面3個function都會跟著一起變動

區域變數(Local Variable)

假設3個function中有各自的變數

分別為apple，banana，cup

3個變數只能各自存活在各自的function中，而且不同的function不能存取不是自己的變數

在JS中我習慣把區域變數的作用範圍（Scope）分成兩類 function內部 以及{大括弧}內部

[補充]：作用在{大括弧}內部就是所謂的作用在block內

Var宣告 vs let宣告

  • Var宣告就是作用在function內部
  • let宣告就是作用在{}內部

Const宣告

Const宣告就是宣告一個常數（儘量不要更動的定值）

Const宣告時一定要給它一個值而不能只給它變數名稱，不然會報錯喔！！！
